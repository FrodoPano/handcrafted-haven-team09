## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/FrodoPano/handcrafted-haven-team09.git

# Navigate to project directory
cd handcrafted-haven-team09

# Install dependencies
npm install

# Run development server
npm run dev
